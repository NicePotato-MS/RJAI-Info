Privacy Policy for RJAI

Last Updated: March 11th, 2025

1. Introduction

Welcome to RJAI, a Discord bot designed to roleplay with users using a Large Language Model (LLM). This Privacy Policy explains how we collect, use, and protect user data while ensuring compliance with relevant regulations. By using RJAI, you agree to the terms outlined below.

2. Data Collection

2.1 Temporarily Collected Data

  • Messages sent by users (used to generate responses via the LLM).
  • Information accessible via Discord's API (such as usernames, user IDs, and server IDs).

This data is stored in RAM only and is cleared when the bot restarts or when a certain number of messages fill up the cache.

2.2 Permanently Stored Data

  • User IDs, usernames, and other basic account information.
  • User-specific settings, preferences, and permissions.

This data is not shared with any third party and is only accessible by RJAI’s developers.

2.3 Debugging and Analytics

For debugging and analytics, certain messages may be temporarily logged. These logs are not permanently stored and are used strictly for improving RJAI’s functionality.

3. Data Usage and Sharing

  • No private or sensitive user data is stored, shared, or sold.
  • Any data collected is only used for the bot’s functionality, debugging, and improving user experience.
  • RJAI does not share data with third parties beyond what is already accessible under Discord’s platform.

4. Security Measures

  • The database is password-protected and has no external network access outside of our internal API.
  • Stored data is only accessible by RJAI’s developers and is not shared with any unauthorized third party.

5. User Rights and Controls

  • Commands are available to view and modify stored user data.
  • No private or sensitive data is stored, so data deletion requests are not required.

6. Compliance and Age Restrictions

  • RJAI complies with all applicable laws, regulations, and Discord's Terms of Service.
  • Users under 13 years old are strictly prohibited from using RJAI. If a user is found to be underage, they will be banned from using the bot.
